### Getting started


Checkout [docs](https://itsjwala.github.io/locus)



### Developing

1. install docker on the machine

2. start locus

```sh
make run 
```

1. To Stop the container send SIGTERM `ctrl ^ c`


2. clear the images that were created

```sh
make clear
```
