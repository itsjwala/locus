### Getting started


Checkout [Document Link](https://itsjwala.github.io/locus)

### Instructions

1. install docker engine on the machine

2. start locus

```sh
make run 
```

### Clean Up

1. To Stop the container send SIGTERM `ctrl ^ c`


2. clear the images that were created

```sh
make clear
```
