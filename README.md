## Usage


### Prerequisite

1. Install [Docker](https://docs.docker.com/engine/install/) ( thats it :smile: )

### Installation and Setup

1. start locus 

```sh
make run 
```

2. To Stop the web container send SIGTERM `ctrl ^ c`


3. clear the images that were created 

```sh
make clear
```
